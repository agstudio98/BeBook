# User Stories - BeBook Backlog

## Epic 1: Authentication & Profile
- **US-1:** As a new visitor, I want to create an account so that I can access personalized features.
- **US-2:** As a user, I want to login with my Google account for a faster and more secure experience.
- **US-3:** As a client, I want to update my profile picture and subscription plan to keep my information current.

## Epic 2: Catalog & Shopping
- **US-4:** As a reader, I want to filter books by category and type so that I can find my next read easily.
- **US-5:** As a customer, I want to manage my shopping cart so that I can review my items before purchasing.
- **US-6:** As a customer, I want to pay for my books securely using different payment methods.

## Epic 3: Community & Interaction
- **US-7:** As a forum user, I want to start a new discussion thread to share my opinions on a book.
- **US-8:** As a reader, I want to review and rate products to help other community members.
- **US-9:** As a user, I want to see the latest activities and slogans on the home page to stay engaged.

## Epic 4: Branch Services
- **US-10:** As a user, I want to find the nearest BeBook branch using my location to visit it in person.
- **US-11:** As a client, I want to book a study table or an event through a calendar interface.

## Epic 5: Administration
- **US-12:** As an administrator, I want to manage the product inventory to ensure the catalog is up to date.
- **US-13:** As an administrator, I want to oversee forum content to maintain a respectful community environment.
